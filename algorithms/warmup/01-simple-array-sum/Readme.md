# Simple Array Sum

## Task
Given an array of integers, can you find the sum of its elements?

#### Sample Input

```
6
1 2 3 4 10 11
```

#### Sample Output

```
31
```

#### Instructions

Type `make` to build.

Type `make run` to run and insert input manually.

Type `make runinput` to run and get input from `etc/input` file.
