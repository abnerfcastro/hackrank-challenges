# Day 5: Loops

## Task

Refer to [link](https://www.hackerrank.com/challenges/30-loops) for official instructions.

### Sample Input

```
2
```

### Sample Output

```
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
2 x 4 = 8
2 x 5 = 10
2 x 6 = 12
2 x 7 = 14
2 x 8 = 16
2 x 9 = 18
2 x 10 = 20
```

#### Instructions

Type `make` to build.

Type `make run` to run and insert input manually.

Type `make runinput` to run and get input from `etc/input` file.
