# Day 7: Arrays

## Task

Refer to [link](https://www.hackerrank.com/challenges/30-arrays) for official instructions.

### Sample Input

```
4
1 4 3 2
```

### Sample Output

```
2 3 4 1
```

#### Instructions

Type `make` to build.

Type `make run` to run and insert input manually.

Type `make runinput` to run and get input from `etc/input` file.
